# ICP and Messaging Guardrails

## Public ICP Framing
- Use natural public phrasing: "for U.S. businesses running multiple crews."
- Do not force strict internal qualifier language into public SEO copy.
- Keep exact numeric qualifiers as internal strategy references only.

## Core Promise Order
Use this sequence consistently in primary sections:
1. Reduce payroll overpayment.
2. Strengthen compliance confidence.
3. Cut weekly payroll admin time.

## Voice Rules
- Headline style: founder-direct and outcome-led.
- Body style: professional, proof-led, and operationally specific.
- Avoid fluffy claims without implementation context.

## Claims Policy
- Conversion-forward numeric claims are allowed when tied to realistic context.
- Avoid unsupported precision claims that imply guaranteed outcomes for every team.

## Red Flags
- Disallowed public qualifier phrases:
  - "11-50 employees"
  - "$500k-$5M"
- Promise sequence presented in a different order on core SEO templates.
